VPPMapCustomAnnotation Protocol Reference

Conforms to MKAnnotation
Declared in VPPMapCustomAnnotation.h

Overview

This protocol can be used to customize an annotation.

VPPMapHelper will detect if a MKAnnotation conforms to VPPMapCustomAnnotation protocol, and in that case would read its customization.
